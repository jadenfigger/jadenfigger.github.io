Past Projects

2048 Clone in HTML/Javascript

A 2048 clone created in HTML with Javascript and CSS. The clone has all the features of the original game, such as a score/best score counter. The game uses the local storage, allowing browser to remember the best score between visits. In future versions of the clone, I plan on creating an AI to play the game, that could achieve a score higher than any human normally could.

Fantasy Football Manager

A Winforms application that allows the user to manage an NFL fantasy football leauge. This includes creating teams, adding players to those teams, creating a schedule to matchup teams against each other. Then you can, with the press of a button, calculate point totals for those teams for the current week. The program also allows you to view each fantasy week to see stats about the teams. Some of these stats include each teams record, average points scored per week, total points scored, and much more.

Tic-Tac-Toe w/ Computer AI

Created in python with the game building library, Pygame, I created a Tic-Tac-Toe clone with a fully function enemy AI and an adjustable board size. The AI uses the minimax algorithm, which simulates all possible future moves to find the best next move. The game board is adjustable, allowing the user to change the game from 3x3 all the way up to 21x21.
